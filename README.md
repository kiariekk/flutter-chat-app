## Getting Started

Note: You already need to have Flutter installed.

<pre>
# clone the repo
git clone https://github.com/marciovalim/flutter-chat-app.git 

# move to the cloned repo
cd flutter-chat-app

# get dependencies
flutter pub get

# run application
flutter run
</pre>
